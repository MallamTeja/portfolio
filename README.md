# Teja Mallam Portfolio

My portfolio website developed with Next.JS(SSG) and TypeScript. Tailwind CSS and GSAP is used for styling and animations.

**View my portfolio via the link in the repository's About section.**

## Features

- Interactive UX
- Responsive Design 📱💻
- Light & Dark themes ☀️🌙
- Fully Accessible
- SEO Friendly 🔎
  -deployed in custom domain(tejamallam.tech) for both ipv4 and ipv6 and accesble with both apex domain(@) and third level domain(www)

## Tech Stack

**Frontend** - [NextJS](https://nextjs.org/), [React](https://reactjs.org/), [TypeScript](https://www.typescriptlang.org/)  
**Styling** - [Tailwind CSS](https://tailwindcss.com/)  
**Animations** - [GSAP](https://greensock.com/)  
**UIUX** - [Figma](https://figma.com/)  
**State Management** - [ContextAPI](https://reactjs.org/docs/context.html)  
**Backend** - [Forestry Headless CMS](https://forestry.io/)  
**Deployment** - [Vercel](tejamallam.me)

## Running Locally

Clone the project

```bash
git clone https://github.com/tejamallam/tejamallam.dev.git
```

Install dependencies

```bash
npm install
```

Start the server

For Dev environment

```bash
npm run dev
```

For normal environment

```bash
npm start
```
